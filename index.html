HTML tags

TagNameDescription
tableTableThe wrapper element for all HTML tables
theadTable HeadThe element defines a set of rows defining the head of the columns of the table.
tbodyTable BodyThe element encapsulates a set of table rows.
trTable RowThe element defines a row of cells in a table.
tdTable DataThe element defines a cell of a table that contains data.
tfootTable FootThe set of rows defining the footer in a table.

HTML attributes

TagNameDescription
colspanColumn SpanDefines how many columns a td element should span.
rowspanRow SpanDefines how many rows a td element should span.

CSS Styling

PropertyDescription
border-collapse: separate;Sets whether table borders should collapse into a single border or be separated.
border-spacing: 5px;When using "border-collapse: separate", the border-spacing property can be used to set the space between the cells.
